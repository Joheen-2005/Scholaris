# Scholaris

Scholaris is a multi-agent AI-powered research intelligence system built with Streamlit and integrated with Groq-hosted large language model (Llama 3.1). It assists researchers with literature mining, trend analysis, research gap identification, methodology design, IEEE paper drafting, grant proposal generation, citation formatting (APA/MLA/IEEE), and automated novelty scoring. Featuring a creativity control slider and a futuristic, interactive UI, Scholaris acts as an intelligent academic co-pilot that helps transform research ideas into structured, high-quality outputs efficiently.
